Ribcage Button
==============

A little view to use with `ribcage-view` that gives you a button.

## Install

```
npm install ribcage-button
```

## Usage

```javascript

  var Button = require('ribcage-button')

  var button = new Button({
    label: 'Submit'
  , action: function () {
      alert('submitted')
    }
  })

```
